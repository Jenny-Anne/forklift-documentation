= Resolved issues 2.11.6

[role="_abstract"]
Review the resolved issues in this release of {project-short}. Some linked Jira tickets are accessible only with Red Hat credentials.

Migrations no longer fail unexpectedly due to memory issues on ESXi hosts::
Before this update, the populator automatically installed the vmkfstools-wrapper VIB on every clone operation. As a consequence, memory issues often occurred on ESXi hosts, which caused migrations to fail unexpectedly. With this release, the automatic VIB installation feature is removed from the populator. As a result, the memory-related failures no longer occur during clone operations. Users who require the vmkfstools-wrapper VIB must now install it manually before they initiate a migration.
